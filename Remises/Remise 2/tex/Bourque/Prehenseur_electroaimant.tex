Afin de conserver l’énergie dans le condensateur le plus longtemps possible, le courant de décharge dans l’électroaimant est limité par un MOSFET. Une onde carré de largeur d’impulsion variable est envoyée dans sa base et permet de contrôler ce courant. 
\medbreak
Avec un condensateur de 5F chargé à 5V et du limiteur de courant, l'électroaimant peut soulever un trésor durant 3 ½ minutes, ce qui est largement suffisant pour le projet.
\medbreak
Le système comprend aussi un interrupteur électronique afin de permettre de couper la décharge de courant dans l’électroaimant et ainsi déposer le trésor.
